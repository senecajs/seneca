Seneca

A Micro-Services toolkit for Node.js

Download and Install

Current Version: 0.6.3

Seneca is distributed as an npm module. To download and install, open a terminal and go to your project folder. Run the command:

$ npm install seneca

If you have set up a package.json file, then run this command to add Seneca to your dependencies:

$ npm install seneca --save

In your code, you can import the Seneca module, and create a new object instance, with this line of code:

>var seneca = require('seneca')()

Note that require('seneca') provides a function that creates a Seneca instance. This is the same as the way you create a new Express instance.

If you're using Seneca to create microservices, your service start-up code will often look like this:

← load Seneca module, and create instance
← register a service definition as a Seneca plugin
← listen for service clients on port 8001
require('seneca')()
  .use('my-service')
  .listen(8001)

Read the Getting Started Guide to start using Seneca in your app.

You can also clone Seneca directly from GitHub: github.com/rjrodger/seneca. Do this if you need to use a development branch.
