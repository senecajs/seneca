Action Based API

Seneca is built from the ground up as an action based toolkit. Map your requirements straight to business logic and use Seneca's expressive filtering syntax to glue microservices together with ease.

Pluggable Transport

Transport and storage concerns are handled in Seneca as plugins. Use TCP for one service and HTTP for another. Our storage api allows data storage to be handled on a per entity basis so you can use the best solution without Seneca getting in the way.

Storage Neutral

Seneca provides a lean core, but is massively extensible via plugins. We provide a vast array of in house plugins and also list any community ones in our plugin section. We have many of the basic plugins already covered and provide an excelent guide for building your own.
